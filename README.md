# Smart Resume Analyzer

A web-based Resume Analyzer built using Python and Streamlit.

## Features
- Upload resume (PDF)
- Extract resume content
- Detect key skills
- Resume scoring system
- Improvement suggestions
- Job description matching

## Technologies Used
- Python
- Streamlit
- pdfplumber

## How to Run
pip install streamlit pdfplumber  
python -m streamlit run app.py

## Purpose
This project helps users evaluate their resumes and improve them based on job requirements.# Smart-Resume-Analyzer
Developed a web-based Resume Analyzer using Python and Streamlit that extracts resume content, detects key skills, evaluates resumes using a scoring system, and provides improvement suggestions. Implemented a job description matching feature to calculate match percentage and identify missing skills for better job alignment.
